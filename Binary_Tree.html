About Binary Tree

Learn about the operations, implementation, and time complexity of About Binary Tree.

Binary Tree Operations

  1. A Binary Tree is a hierarchical data structure in which each node has at most two children:
    • Left Child
    • Right Child

    Example:

    
        A  ------> Level 1 (A)
       / \
      B   C   -------> Level 2(B, C)
     / \   \
    D   E   F ----------> Level 3(D, E, F)

    • A parent node can have up to two children: Left Child and Right Child

    • In the above example, if we consider A as the parent node:
      • B is the Left Child of A
      • C is the Right Child of A

    • Similarly, if we consider B as the parent node:
      • D is the Left Child of B
      • E is the Right Child of B

    • If we consider C as the parent node:
      • F is the Right Child of C

Node Structure 
struct Node
{
    int data;
    struct Node *left;
    struct Node *right;
};
1. Create a node
2. Store data in node
3. Initialize left and right child pointers as NULL
No Time Complexity
Create a new node
Algorithm
1. Allocate memory for node
2. Store data
3. Set Left = NULL
4. Set Right = NULL
5. Return Node
Code 
struct Node* createNode(int data)
{
    struct Node *newNode =
        (struct Node*)malloc(sizeof(struct Node));

    newNode->data = data;
    newNode->left = NULL;
    newNode->right = NULL;

    return newNode;
}
 O(1) or O(n)
Insertion
Algorithm
1. Create a new node
2. If tree empty:
Make it root
3. Otherwise:
Traverse level by level
Insert at first empty position
Code 
struct Node* insert(struct Node *root, int data)
{
    if(root == NULL)
        return createNode(data);

    if(data < root->data)
        root->left = insert(root->left, data);
    else
        root->right = insert(root->right, data);

    return root;
}
 O(log n)
Search
Algorithm
1. Compare key with root
2. If equal then print found
3. If smaller then search left sub tree
4. If greater then search right sub tree
5. Repeat untill found or null
Code 
struct Node* search(struct Node *root, int key)
{
    if(root == NULL || root->data == key)
        return root;

    if(key < root->data)
        return search(root->left, key);

    return search(root->right, key);
}
 O(log n)
Inorder Traversal
(Left -> Root-> Right)

First visit Left Subtree, then Root, then Right Subtree
Algorithm
1. Traverse Left Subtree
2. Visit Root
3. Traverse Right Subtree
Code 
void inorder(struct Node *root)
{
    if(root != NULL)
    {
        inorder(root->left);
        printf("%d ", root->data);
        inorder(root->right);
    }
}
 O(n)
Preorder Traversal
(Root -> Left-> Right)

First visit Root, then Left Subtree, then Right Subtree
Algorithm
1. First visit Root
2. Traverse Left Subtree
3. Traverse Right Subtree
Code 
void preorder(struct Node *root)
{
    if(root != NULL)
    {
        printf("%d ", root->data);
        preorder(root->left);
        preorder(root->right);
    }
}
O(n)
Postorder Traversal
(Left -> Right -> Root)

First visit Left Subtree, then Right Subtree, then Root
Algorithm
1. Traverse Left Subtree
2. Traverse Right Subtree
3. Visit Root
Code 
void postorder(struct Node *root)
{
    if(root != NULL)
    {
        postorder(root->left);
        postorder(root->right);
        printf("%d ", root->data);
    }
}
O(n)
Min element in Binary Tree
Algorithm
1. Move continously to Left Child
2. Last Left node is minimum
Code 
struct Node* findMin(struct Node *root)
{
    while(root->left != NULL)
        root = root->left;

    return root;
}
O(log n)
Max Element in Binary Tree
Algorithm
1. Move continously to the right child
2. Last right node is maximum
Code 
struct Node* findMax(struct Node *root)
{
    while(root->right != NULL)
        root = root->right;

    return root;
}
O(log n)
Deletion of a node
Algorithm
1. If you want to delete Leaf Node, Delete directly
2. If you want to delete One Child, Replace node with child
3. If you want to delete Two Children, Replace with inorder successor and Delete successor
Code 
struct Node* deleteNode(struct Node *root, int key)
{
    if(root == NULL)
        return root;

    if(key < root->data)
        root->left = deleteNode(root->left, key);

    else if(key > root->data)
        root->right = deleteNode(root->right, key);

    else
    {
        if(root->left == NULL)
        {
            struct Node *temp = root->right;
            free(root);
            return temp;
        }

        else if(root->right == NULL)
        {
            struct Node *temp = root->left;
            free(root);
            return temp;
        }

        struct Node *temp =
            findMin(root->right);

        root->data = temp->data;

        root->right =
            deleteNode(root->right,
                       temp->data);
    }

    return root;
}
O(log n)
